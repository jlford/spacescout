For Developers

Get the code

All SpaceScout projects are hosted on GitHub.

  • SpaceScout builds is a sample project template to help you get up and running
  • SpaceScout server is required to get started
  • SpaceScout web app
  • SpaceScout iOS app
  • SpaceScout LabStats integrates with LabStats software installed in your computer labs to show students real-time computer availability in SpaceScout

Get it running

See Spacescout Dev Installation on the spacescout_builds wiki for step-by-step instructions on setting up a developer installation with our project templates.

Submitting contributions

For now, if you have contributions, just submit a pull request via GitHub. Please include unit tests if you can.
